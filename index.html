⚙️ Google OAuth Setup Guide
  1. Go to console.cloud.google.com → Create a new project (e.g. "Ledger")
  2. Go to APIs & Services → Library → Search for Google Drive API → Enable it
  3. Go to APIs & Services → OAuth consent screen → External → Fill in app name → Add your Gmail as a test user
  4. Go to APIs & Services → Credentials → Create Credentials → OAuth 2.0 Client ID → Web application
  5. Under Authorized JavaScript origins, add:
    https://YOUR_USERNAME.github.io
  6. Copy the Client ID (looks like: xxxxxxxxxx.apps.googleusercontent.com)
  7. Open index.html → find YOUR_GOOGLE_CLIENT_ID → replace with your Client ID
  8. Push to GitHub → enable GitHub Pages → done! ✅
